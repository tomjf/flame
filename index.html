flame

Flame data reduction pipeline

Flame is a flexible data reduction pipeline for the LUCI instrument at the Large Binocular Telescope, written by Sirio Belli (MPE)

To install: download the flame/ directory and add it to your IDL path. The latest versions of the NASA IDL library, Coyote library, and mpfit library are required.

The user manual is in docs/manual.pdf
